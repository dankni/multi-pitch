About this application

This app is based on the metoliusclimbing.com plan for use with their rock rings. We are not in any way affiliated with Metolius. This application was built for personal use but is free to use for anyone that finds it helpful. The code is also open scourced to anyone who wants to use, copy or contribute to it. Be warned, it was made in a couple of hours last weekend (Jan 2022) and the code quality reflects that. Github Page
